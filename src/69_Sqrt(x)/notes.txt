haskel version
```
sqrt x = sqrt' x 1.0 
  where 
    sqrt' x guess = if good_enough (abs ((guess * guess) - x))
	                then guess
    	            else sqrt' x (improve guess)
    good_enough = (>) 0.001
    improve guess = ((x / guess) + guess ) / 2 
```
